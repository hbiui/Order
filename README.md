# 家味点餐 - 家庭点餐系统

一个温馨的家庭点餐系统，支持家庭成员管理、余额与家务劳动双重支付机制、菜品管理及订单状态追踪，专为移动端优化设计。

## 🚀 功能特性

- 👨‍👩‍👧 **家庭成员管理**：添加/编辑家庭成员，设置权限和余额
- 🍽️ **菜品管理**：丰富的菜品库，支持口味选择和分类
- 💰 **双重支付**：支持余额支付和家务劳动点数支付
- 🛒 **智能购物车**：实时计算总价，支持多种支付方式混合
- 📋 **订单管理**：批量订单管理，状态跟踪
- 📱 **移动优先**：专为移动端优化的UI设计
- 🔄 **本地存储**：所有数据本地保存，无需后端
